Practical 1: Network Monitoring Technologies

1A. SPAN Technologies (Switch Port Analyzer)

Aim: Implement SPAN Technologies to monitor switch port traffic.

Packet Tracer Setup Instructions:

  1. Drag and drop 1 Router (2911), 1 Switch (2960), and 1 PC from the device panel
  2. Connect PC0 to Switch0 Fa0/1 using copper straight-through cable
  3. Connect Switch0 Fa0/2 to Router R1 Fa0/0 using copper straight-through cable
  4. Configure IP addresses as per addressing table

Addressing Table:

Switch0   Fa0/1        N/A           N/A            N/A

Custom Topology Script

from mininet.topo import Topo
from mininet.net import Mininet
from mininet.node import RemoteController
from mininet.cli import CLI
from mininet.log import setLogLevel

class CustomTopo(Topo):
    def build(self):
        # Add switches
        s1 = self.addSwitch('s1')
        s2 = self.addSwitch('s2')
        s3 = self.addSwitch('s3')
        
        # Add hosts
        h1 = self.addHost('h1', ip='10.0.0.1/24')
        h2 = self.addHost('h2', ip='10.0.0.2/24')
        h3 = self.addHost('h3', ip='10.0.0.3/24')
        h4 = self.addHost('h4', ip='10.0.0.4/24')
        
        # Add links
        self.addLink(h1, s1)
        self.addLink(h2, s1)
        self.addLink(h3, s2)
        self.addLink(h4, s3)
        self.addLink(s1, s2)
        self.addLink(s2, s3)
        self.addLink(s3, s1)

def runCustomTopo():
    topo = CustomTopo()
    net = Mininet(topo=topo, controller=RemoteController)
    net.start()
